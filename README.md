# WebAssembly-OpenGL

# Welcome

Creating project with SDL and OpenGL using Emscripten/WebAssembly ! 

## Run Locally

Clone the project

```bash
  git clone https://github.com/KorieDrakeChaney/WebAssembly-OpenGL.git
```

Go to the project directory

```bash
  cd WebAssembly-OpenGL
```

Start 

```bash
  npm run start
```
